# 🚢 Titanic Mini Visualization Dashboard

A complete **Mini Data Visualization Dashboard** built using **Python, Pandas, Matplotlib, and Seaborn** on the Titanic dataset. This project focuses on data cleaning, feature engineering, exploratory data analysis (EDA), and visual storytelling through multiple charts.

This project was developed as **Task 4** of the **Data Analysis with Python Internship** at **Maincrafts Technology**.

---

# 📌 Project Objective

The objective of this project is to analyze the Titanic dataset and communicate meaningful insights through clear and effective visualizations.

The project demonstrates:

- Data Cleaning
- Handling Missing Values
- Feature Engineering
- Exploratory Data Analysis (EDA)
- Data Visualization
- Insight Generation

# ⚙ Data Preprocessing

The following preprocessing steps were performed:

- Loaded Titanic Dataset
- Checked Dataset Information
- Identified Missing Values
- Filled Missing Age values using Mean
- Filled Missing Embarked values using Mode
- Removed Cabin column
- Created Family Size feature
- Created Age Group feature

# 📈 Key Insights

Some important observations obtained from the analysis:

- Female passengers had a significantly higher survival rate than male passengers.
- First Class passengers had the highest survival probability.
- Most passengers belonged to the age group of 20–40 years.
- Small family groups had better survival chances.
- Fare and Passenger Class showed a noticeable relationship with survival.
- Embarkation location influenced passenger survival rates.
